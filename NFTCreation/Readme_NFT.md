## Идея 1 для Utility NFT
С помощью NFT-коллекции можно организовывать различные соревнования, игры, учебные задания с набором очков и на время.
**Примеры:**

[1] В соревновательной задаче по программированию какого-либо приложения предусмотрена поэтапная разработка с постепенным наращиванием функционала – n этапов.
Ведущий (преподаватель или тьютор) создает коллекцию NFT из n лотов различной стоимости, причем:

- количество лотов равно количеству усложнений в задаче;
- стоимость лотов рассчитывается в зависимости от уровня сложности задания очередного этапа.

По мере решения участник, первым правильно выполнивший задание очередного этапа, получает соответствующий лот данного NFT.

Поскольку лоты уникальные и невзаимозаменяемые, а также имеют свою стоимость, то:
- во-первых, ведущий имеет четкие доказательства авторства (по отслеживаемым транзакциям);
- а во-вторых, по общей стоимости заработанных лотов участника автоматически генерируется оценка за выполнение задания.

[2] Аналогично примеру [1] можно организовать контрольные работы и олимпиады по математике, программированию, языкам и т. п.

[3] В развлечениях типа квестов для успешного завершения игры необходимо выполнить k заданий, причем каждое последующее выполняется только после предыдущего.
Ведущий создает коллекцию NFT из k лотов одинаковой стоимости. Участники заказывают игру и резервируют время, купив 1-й лот NFT-коллекции. Его стоимость является предоплатой и определяется организатором игры.
Участники приступают к игре, начав с покупки второго лота.
- Количество заданий в квесте на 2 меньше количества лотов, то есть заданий k-2 штук.
- Каждый из k-2 лотов содержит необходимую подсказку для решения следующего задания (текст, записанное аудио, видео, карта местности и т. п.).
- Дополнительный лот в коллекции не обязателен для приобретения, но он дает право на дополнительную подсказку. Его можно купить на любом этапе игры. И он стоит несколько больше, чем остальные лоты.
- Прибыль от покупки лотов во время игры – это оставшаяся плата за организацию игры (помимо предоплаты, внесенной посредством покупки первого лота).

При организации игры с помощью NFT обеспечиваются следующие преимущества:
- Гибкая система оплаты. На любом этапе игроки могут прекратить игру, заплатив только за те задания, которые ими решались. И организатор получает возмещение за использованный реквизит (который, возможно, требуется обновлять после каждой игры).
- Игроки получают уникальное доказательство победы в квесте в виде собранной коллекции NFT.

## Идея 2 для Utility NFT
Регулирование долей авторства в групповом проекте, особенно, если рассчитывается вознаграждение по размеру вклада в проект. Это могут быть акции предприятий, главы в книгах, разработанные модули в проектах, вклады в общее дело.

## Идея 3 для Utility NFT
Организация бартера цифровым контентом, в том числе покупка услуг. Покупая лот какой-либо NFT-коллекции, участник приобретает реферат, компьютерную программу, музыку, сочинение, решение задачи и т. п.

## Идея 4 для Utility NFT (реализован прототип в одной из задач EXTRA)
Децентрализованный портал для развлечений.
**Пример.** «Загадки для детективов». Желающие могут поупражняться в дедукции, отвечая на вопросы к изображениям. Utility NFT-коллекция – https://testnets.opensea.io/collection/riddlesfordetectives.

Ответы и объяснения получают участники, купившие лот с загадкой. Первый давший правильный и полный ответ, получает лот, дающий право на скидку на следующий лот с загадкой; остальные получают копии лота.

Желательно реализовать dApp-приложение, в котором можно выбирать загадки по изображениям, по описаниям, и сразу после покупки доступны ответы (без необходимости отправлять запрос).

Также в dApp должны быть реализованы аккаунты, вестись счет, реализовано присваивание уровней проницательности, скидки постоянным пользователям, базы развлечений должны регулярно пополняться (посредством регулярной чеканки новых NFT).
